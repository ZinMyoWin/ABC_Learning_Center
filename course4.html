HTML

HTML is the foundation of web development, and this course teaches students how to create websites using HTML. Students will learn how to structure web pages, use tags and attributes, and add multimedia elements. The course is taught by Ms.Cardi B and starts on Fridays from 6 pm to 9 pm. It takes 4 weeks to complete the course.
The course outcomes include the ability to design and create websites using HTML and an understanding of web development concepts.
