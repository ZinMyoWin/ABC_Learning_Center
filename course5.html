Web Development with JavaScript

JavaScript is a programming language used for creating dynamic web applications. This course teaches students how to use JavaScript to create interactive web pages and build web applications. Students will learn how to manipulate the Document Object Model (DOM), use events and functions, and create AJAX requests. The course is taught by Ms.Taylor Swift and starts on Mondays and Wednesdays from 7 pm to 9 pm. It takes 12 weeks to complete the course.

Learning outcomes

The course outcomes include proficiency in using JavaScript for web development and an understanding of web application architecture.
