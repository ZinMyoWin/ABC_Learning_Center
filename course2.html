Programming in .Net

This course covers the basics of the .Net framework and how to use it for web application development. Students will learn how to use the C# programming language, create web forms, and work with databases using ADO.Net. The course is taught by Ms.Taylor Swift and starts on Tuesdays and Thursdays from 7 pm to 9 pm. It takes 10 weeks to complete the course.
The course outcomes include the ability to create .Net applications, understand the .Net framework, and use Visual Studio for development.
